1. Просмотр расписания

      Для просмотра расписания сеансов на кинофильм необходимо:

    1. Нажать на постер желаемого кинофильма.

    2. На открывшейся странице фильма выбрать нужные кинотеатр, кинозал (опционально), день, кликнув на соответствующие кнопки. Внизу отобразится время начала доступных сеансов.

      3. Просмотр расписания доступен как авторизованным, так и неавторизованным пользователям. Другие функции доступны только авторизованным пользователям!

  2. Регистрация

      Для того, чтобы зарегистрироваться в системе, необходимо:

    1. Нажать на кнопку "Регистрация" в правом верхнем углу экрана.

    2. На открывшейся странице регистрации заполнить поля формы.

    3. После заполнения полей, нажать на кнопку "Зарегистрироваться".

  3. Авторизация

      Для того, чтобы авторизоваться в системе, необходимо:

    1. Нажать на кнопку "Вход" в правом верхнем углу экрана.

    2. На открывшейся странице авторизации заполнить поля формы.

    3. После заполнения полей, нажать на кнопку "Зарегистрироваться".

  4. Бронирование билетов

      Для бронирования билетов, необходимо:

    1. Повторить действия из пункта "Просмотр расписания", предварительно авторизовавшись.

    2. Выбрать подходящее время сеанса из списка доступных, кликнув по соответствующей кнопке.

    3. Кликнуть на кнопку "Далее" в появившемся окне

    4. На странице выбора места, выбрать нужные места, кликнув на них и нажать на кнопку "Забронировать".

    5. Далее система откроет страницу подтверждения заказа. Для подтверждения заказа нажмите кнопку "Забронировать".

    6. Если бронирование прошло успешно, система откроет страницу со статусом бронирования

  5. Просмотр билетов

      Для того, чтобы просмотреть имеющиеся билеты, необходимо:

    1. Предварительно авторизоваться/зарегистрироваться в системе (см. пункты II, III).

    2. Забронировать билет на сеанс (см. пункт IV).

    3. На главной странице нажать на кнопку "Мои билеты"

    4. Система откроет страницу с информацией о билетах на предстоящие сеансы

    5. Для просмотра информации о билетах на прошедшие сеансы необходимо кликнуть на кнопку "История"

  6. Скачать билеты

      Для того, чтобы скачать билет на сеанс, необходимо:

    1. Повторить действия 1-4 из пункта "Просмотр билетов"
      (скачивание билетов доступно только для предстоящих сеансов)

    2. На странице просмотра информации о билетах предстоящие сеансы,
      нажать на кнопку "Скачать билет" нужного билета.

    3. Во вкладке "Загрузки" браузера появится PDF-файл выбранного билета
